### Added
- 75+ LitElement web components with `flint-` prefix
- React wrappers via `@lit/react` (`flint-ui-react`)
- Full theming system with `--flint-*` CSS custom properties
- Built-in dark mode (`flint-theme-dark` class)
- VitePress documentation site with component API reference
- Storybook playground with interactive examples
- 99.4% Vitest unit test coverage
- 80% Stryker mutation testing score
- Accessibility documentation for all components
- Browser support matrix (Chrome 90+, Edge 90+, Firefox 90+, Safari 15.4+)
- CI pipeline with lint, test, and build jobs
- CONTRIBUTING guide with development workflow
